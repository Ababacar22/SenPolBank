# 🤖 SenPolBank — Fraud Detection AI

Module d'intelligence artificielle pour la détection de fraudes.

## Responsabilités

- Scoring automatique des signalements
- Détection de patterns frauduleux
- Priorisation des cas urgents
- Suggestions aux agents de police

## Stack

- **Framework** : Python + FastAPI
- **ML** : scikit-learn / TensorFlow (à définir)
- **Base de données** : PostgreSQL (lecture seule)
