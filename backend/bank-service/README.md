# 🏦 Bank Service

Service d'intégration bancaire de SenPolBank.

## Responsabilités

- Réception des demandes de blocage
- Simulation d'API bancaire (environnement de test)
- Blocage / déblocage de comptes
- Réponse et accusé de réception

## Stack

- **Framework** : NestJS
- **Base de données** : PostgreSQL

## Sprint d'implémentation

Ce service sera développé dans le **Sprint 6** (Bank Integration).
