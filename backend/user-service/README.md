# 👤 User Service

Service de gestion des profils utilisateurs de SenPolBank.

## Responsabilités

- CRUD des profils utilisateurs
- Gestion des rôles et permissions
- Vérification d'identité (KYC simplifié)
- Historique des actions utilisateur

## Stack

- **Framework** : NestJS
- **Base de données** : PostgreSQL

## Sprint d'implémentation

Ce service sera développé dans le **Sprint 2** (User Management).
