# 👮 Police Service

Module de validation policière de SenPolBank.

## Responsabilités

- Réception des signalements de fraude
- Validation ou rejet des plaintes
- Déclenchement du bouton d'urgence (action critique)
- Génération de rapports officiels

## Stack

- **Framework** : NestJS
- **Base de données** : PostgreSQL

## Sprint d'implémentation

Ce service sera développé dans le **Sprint 4** (Police Module).
