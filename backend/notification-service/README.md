# 🔔 Notification Service

Service de notifications de SenPolBank.

## Responsabilités

- Envoi d'emails (signalement reçu, validé, bloqué)
- Notifications temps réel via WebSocket
- File d'attente de notifications
- Templates de messages

## Stack

- **Framework** : NestJS
- **Email** : Nodemailer / SendGrid
- **Temps réel** : Socket.io

## Sprint d'implémentation

Ce service sera développé dans le **Sprint 7** (Notifications).
