# 🚨 Fraud Service

Service de gestion des signalements de fraude de SenPolBank.

## Responsabilités

- Création de signalements de fraude par les citoyens
- Stockage des preuves et métadonnées
- Suivi du statut des signalements
- Transmission à la police pour validation

## Stack

- **Framework** : NestJS
- **Base de données** : PostgreSQL
- **Stockage documents** : MongoDB (pièces jointes)

## Sprint d'implémentation

Ce service sera développé dans le **Sprint 3** (Fraud Reporting).
