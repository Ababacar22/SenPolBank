# 📊 Audit Service

Service d'audit et de traçabilité de SenPolBank.

## Responsabilités

- Enregistrement de toutes les actions critiques
- Logs immuables (qui, quoi, quand)
- Conformité réglementaire
- Export des journaux d'audit

## Stack

- **Framework** : NestJS
- **Base de données** : MongoDB (documents d'audit)

## Sprint d'implémentation

Ce service sera développé progressivement à partir du **Sprint 1**, avec des entrées d'audit ajoutées au fur et à mesure.
