# 🖥️ SenPolBank — Desktop App

Application de bureau pour les agents de police et les opérateurs bancaires.

## Responsabilités

- Accès rapide aux signalements urgents
- Notifications système natives
- Mode hors-ligne partiel
- Sécurité renforcée (verrouillage session)

## Stack

- **Framework** : Electron ou Tauri
- **UI** : React + Chakra UI (partagé avec web-app)

## Sprint d'implémentation

Cette application sera développée après le **Sprint 8** (Frontend).
