# 🌐 SenPolBank — Web App

Application frontend de la plateforme SenPolBank.

## Responsabilités

- Interface citoyen : signalement de fraude
- Interface police : validation des plaintes
- Interface banque : gestion des blocages
- Dashboard administrateur

## Stack

- **Framework** : React
- **UI** : Chakra UI
- **State** : React Query + Context API
- **Routing** : React Router

## Sprint d'implémentation

Cette application sera développée dans le **Sprint 8** (Frontend).
